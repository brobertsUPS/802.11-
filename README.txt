802.11-An implementation of the 802.11 WLAN
=======
USING THIS CODE:

TERMINAL
  This code is best used through the terminal. Go to the directory 802.11-/ and run “javac WiFiClient.java” to compile. Then run the code using “java WiFiClient”.
ECLIPSE
  The Eclipse802.11-.zip file is an archive file for Eclipse.  To use in Eclipse, click import ->import existing project into workspace. Then click the "select archive file" radio button and choose the Eclipse802.11-.zip file.
